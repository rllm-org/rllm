# rllm-model-gateway

Lightweight HTTP gateway sitting between an agent and an LLM endpoint. Picks one of two modes at startup:

- **Passthrough** ‚Äî forwards every request to a configured `upstream_url` (OpenAI, Anthropic, Fireworks, vLLM‚Ä¶) and returns the upstream response unchanged.
- **Adapter** ‚Äî calls an in-process async function with a `NormalizedRequest` and returns its `NormalizedResponse` re-shaped into the agent's wire format.

Records every call as a `TraceRecord` in SQLite. Adapter mode optionally captures token-level extras (`prompt_ids`, `completion_ids`, `logprobs`, `prompt_logprobs`, `routing_matrices`) into a separate blob row.

## Quick start

```bash
uv venv --python 3.11
source .venv/bin/activate
uv pip install -e .

# Standalone passthrough to a local vLLM (CLI mode)
rllm-model-gateway --upstream-url http://localhost:8000/v1
# admin and agent API keys are auto-generated and printed at startup
```

## Two-tier auth

Every gateway issues two API keys at startup:

- **`admin_api_key`** ‚Äî the engine uses this. Required for `POST /sessions`, `GET /sessions/{sid}/traces`, `DELETE /sessions/{sid}`, `POST /admin/flush`, etc.
- **`agent_api_key`** ‚Äî the engine hands this to the sandbox/agent. Required for `POST /sessions/{sid}/v1/...` proxy endpoints. Cannot read traces or enumerate sessions.

Auto-generated by default (`secrets.token_urlsafe(32)`); regenerated each restart. Pin them via `GatewayConfig(admin_api_key=..., agent_api_key=...)`, env vars (`RLLM_GATEWAY_ADMIN_KEY`, `RLLM_GATEWAY_AGENT_KEY`), or CLI flags (`--admin-key`, `--agent-key`) if you need stable values.

`/health` is the only public endpoint.

## Engine-side usage

```python
from rllm_model_gateway import (
    GatewayConfig, GatewayClient, create_app, deserialize_extras,
)
import threading, uvicorn

# 1. Build and run the gateway in a background thread.
config = GatewayConfig(
    upstream_url="https://api.openai.com/v1",
    upstream_api_key="sk-‚Ä¶",                # gateway ‚Üí OpenAI auth
    model="gpt-4o-mini",                    # optional: rewrite body.model on every call
)
app = create_app(config)                    # passthrough mode (no adapter)
threading.Thread(
    target=lambda: uvicorn.run(app, host=config.host, port=config.port),
    daemon=True,
).start()

# 2. Create a session.
client = GatewayClient(f"http://{config.host}:{config.port}", api_key=config.admin_api_key)
sid = client.create_session(
    session_id="rollout-42",
    metadata={"task_id": "math-001"},
    sampling_params={"temperature": 0.7, "max_tokens": 1024},
)

# 3. Hand the session URL + agent_api_key to the agent.
agent_url = client.get_session_url(sid)
agent_key = config.agent_api_key

# Agent code:
#     from openai import OpenAI
#     OpenAI(base_url=agent_url, api_key=agent_key).chat.completions.create(...)

# 4. After the rollout, pull traces.
client.flush()
for trace in client.get_session_traces(sid):
    print(trace.endpoint, trace.content, trace.metrics["gateway_latency_ms"])
    extras = client.get_trace_extras(trace.trace_id)
    if extras:
        fmt, data = extras
        decoded = deserialize_extras(fmt, data)
        # decoded["prompt_ids"], decoded["completion_ids"], decoded["logprobs"], ‚Ä¶
```

## Adapter mode

```python
from rllm_model_gateway import (
    GatewayConfig, NormalizedRequest, NormalizedResponse, Usage, create_app,
    AdapterError,
)

async def my_adapter(req: NormalizedRequest) -> NormalizedResponse:
    # Your engine call goes here ‚Äî e.g. forward to vLLM/SGLang/Tinker.
    return NormalizedResponse(
        content="‚Ä¶",
        finish_reason="stop",
        usage=Usage(prompt_tokens=12, completion_tokens=34),
        # Per-trace structured fields:
        extras={                    # large blobs ‚Äî separate trace_extras row
            "prompt_ids":     [...],
            "completion_ids": [...],
            "logprobs":       [...],
        },
        metrics={                   # small floats ‚Äî inline on traces row
            "queue_time_ms":      4.1,
            "kv_cache_hit_rate":  0.93,
        },
        metadata={                  # small strings ‚Äî inline on traces row
            "worker_id":      "vllm-3",
            "model_revision": "abc123",
        },
    )

# Raise AdapterError to return a specific HTTP status to the agent:
#     raise AdapterError("rate limited", status_code=429)

app = create_app(GatewayConfig(model="my-model"), adapter=my_adapter)
```

Adapter is `async def fn(NormalizedRequest) -> NormalizedResponse`. The gateway will refuse to start if you pass a sync function. `n>1` requests are rejected with 400 in adapter mode.

## Adapter-emitted per-trace fields

| Field | Where stored | Use |
|---|---|---|
| `extras: dict[str, Any]` | `trace_extras` blob (msgpack) | Large data: token IDs, logprobs, MoE matrices. Fetched per-trace via `GET /traces/{tid}/extras`. |
| `metrics: dict[str, float]` | `traces.metrics` JSON column (inline) | Numeric measurements you want to aggregate (latency components, cache hit rates). Returned with every `get_session_traces`. |
| `metadata: dict[str, Any]` | `traces.metadata` JSON column (inline) | Free-form non-numeric tags (worker id, backend, model revision). Returned with every `get_session_traces`. |

Gateway always populates `metrics["gateway_latency_ms"]` (end-to-end latency observed by the gateway).

## Configuration

| Option | Default | Notes |
|---|---|---|
| `host` | `127.0.0.1` | Loopback by default; pass `--host 0.0.0.0` for external access. |
| `port` | `9090` | |
| `db_path` | `~/.rllm/gateway.db` | SQLite path. |
| `upstream_url` | None | Required for passthrough. Follow the upstream's SDK convention (`https://api.openai.com/v1`, `https://api.anthropic.com`). |
| `upstream_api_key` | None | Sent on outbound passthrough (gateway ‚Üí upstream). |
| `model` | None | If set, rewrites the request body's `model` on every call. |
| `sampling_params_priority` | `"client"` | `"client"` ‚Äî request wins on conflicts; `"session"` ‚Äî session wins. |
| `max_retries` | `2` | Connection retries on transient `httpx.ConnectError`. |
| `admin_api_key` | auto-gen | Engine-side secret. |
| `agent_api_key` | auto-gen | Sandbox/agent secret. |

CLI flags mirror the config (`--upstream-url`, `--upstream-key`, `--model`, etc.). Env vars: `RLLM_GATEWAY_*`. Or pass `--config gateway.yaml`.

## Endpoints accepted from the agent

| Path | Format |
|---|---|
| `POST /sessions/{sid}/v1/chat/completions` | OpenAI chat completions |
| `POST /sessions/{sid}/v1/completions` | OpenAI legacy completions |
| `POST /sessions/{sid}/v1/responses` | OpenAI Responses API |
| `POST /sessions/{sid}/v1/messages` | Anthropic Messages API |

Sessions must be created with `POST /sessions` first; hitting an unknown session ID returns 404.

## Management endpoints (admin key required)

| Method | Path | Description |
|---|---|---|
| `POST` | `/sessions` | Create a session (`session_id`, `metadata`, `sampling_params`). |
| `GET` | `/sessions` | List sessions. |
| `GET` | `/sessions/{sid}` | Session info. |
| `DELETE` | `/sessions/{sid}` | Delete session and its traces. |
| `GET` | `/sessions/{sid}/traces` | List traces for a session (metadata only). |
| `GET` | `/traces/{tid}` | Single trace. |
| `GET` | `/traces/{tid}/extras` | Trace extras blob (msgpack). |
| `POST` | `/admin/flush` | Drain pending async writes + flush WAL. |
| `GET` | `/health` | Public; for liveness probes. |
